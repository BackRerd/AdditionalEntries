**Additional Entries** is a Minecraft 1.20.1 (Forge) mod that adds a comprehensive affix management system for **ClassBioArsenal (CBA)**.

### ğŸŒŸ Key Features
- **Affix Gems**: Specialized gems for weapons and armor that store and transfer powerful attributes.
- **Anvil Transfer**: Seamlessly inject affixes from gems into your equipment using an Anvil.
- **Reforging System**: Includes "Entry Reforge" and "Quality Reforge" items to reroll gear stats.
- **Smart Commands**: Use the `/affixgem` command with full tab-completion to customize your gems.

### ğŸ› ï¸ Quick Start
1. **Obtain Gems**: Get Affix Gems via Creative mode or gameplay.
2. **Generate Affixes**: Right-click "Creative" gems or use "Auto" gems to get random stats.
3. **Apply to Gear**: Place your equipment in the left Anvil slot and the gem in the right slot.
4. **Reforge**: Use reforge items in an Anvil to perfect your gear's attributes and quality.

### ğŸ“Š Technical Info
- **Version**: 1.20.1 (Forge)
- **Dependency**: ClassBioArsenal (CBA)
- **Config**: Reforge success rates are fully customizable in the config file.
